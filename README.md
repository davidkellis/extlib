# extlib

extlib is a collection of extentions to the Crystal standard library that I find myself using frequently.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     extlib:
       github: davidkellis/extlib
   ```

2. Run `shards install`

## Usage

```crystal
require "extlib"
```
